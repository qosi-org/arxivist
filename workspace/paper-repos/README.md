# U-Net: Convolutional Networks for Biomedical Image Segmentation (arxiv_1505_04597)

Reproduction of U-Net (Ronneberger, Fischer & Brox, MICCAI 2015), generated by ArXivist.

## Contents
- `src/unet.py` — U-Net with unpadded convs and crop-and-concat skip connections (23 conv layers)
- `train.py` — smoke-test loop (SGD, momentum 0.99, batch size 1, 572->388 tiles)
- `configs/config.yaml` — default hyperparameters
- `notebooks/` — reproduction notebook

## Run
```bash
pip install torch pyyaml
python train.py
```

Note: valid convolutions shrink the output; a 572x572 input yields a 388x388x2 map.
