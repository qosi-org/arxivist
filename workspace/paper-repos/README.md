# Batch Normalization — ArXivist Reproduction

Reproduction of Ioffe & Szegedy (2015), "Batch Normalization: Accelerating Deep Network Training
by Reducing Internal Covariate Shift" (arXiv:1502.03167), generated by ArXivist.

## Setup

```bash
pip install torch torchvision pyyaml
python train.py
```

## What this reproduces

The MNIST experiment from Section 4.1: a 3-layer sigmoid MLP with Batch Normalization,
trained for 50,000 steps on MNIST (batch size 60). The paper shows BN networks train
faster and reach higher accuracy than the baseline without BN.

## Paper's reported results

| Metric | Dataset | Paper value |
|---|---|---|
| Top-5 error | ImageNet (BN-Inception ensemble) | 4.9% |
| Top-1 error | ImageNet (BN-Inception ensemble) | 20.1% |
| Training speedup | ImageNet vs Inception | 14x fewer steps |

Note: ImageNet results require the full Inception architecture (not included here).
This repo reproduces the MNIST illustrative experiment from Section 4.1.
